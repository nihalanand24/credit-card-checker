Credit Card Checker

Get test credit card numbers from the following payment processing websites.

Enter any number to check if it is a valid credit card number using the Luhn Algorithm. (see wikipedia)

    This page cannot and will not store your credit card information so it is completely safe. All calculations are completed behind-the-scenes within your browser.

  • If you do not wish to enter your own credit card number - please use the following test numbers, or use the Stripe and PayPal links above for more test numbers
    • MasterCard: 5454 5454 5454 5454
    Visa: 4242 4242 4242 4242
    American Express: 3434 3434 3434 343

    Please Note:

  • Only validates Visa, MasterCard, and American Express numbers.
  • Will not work with 19-digit Visa card numbers
